## Getting Started

First, install the dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

On the Home Page, you can see the list of appointments and a small message indicating that there are no appointments.

On the Doctors Page, there is a list of available doctors. In the header section, there will be a navigation dropdown to filter by specialty. On each doctorâ€™s card, you can book an appointment and choose the desired date.
